# Edge Decal Normals
Custom Normals for Edge Decal Meshes in Blender 2.8

How-To:
  - Select Mesh
  - Go into edit mode
  - Select Verts you want to use for Normal Transfer
  - Toolbar -> Tools -> Edge Decal Normals
